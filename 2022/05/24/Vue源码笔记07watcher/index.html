发表于 2022-05-24 更新于 2022-06-24 分类于 Vue源码 Valine：

Watcher

在Vue中数据是响应式的，但是我们想要的不止于此，我们想要一套系统，可以监听数据的变化，并且通知我们，以便进行后续的处理，这就是watcher。

Watcher和watch的区别

watch是Vue提供给用户的一个自定义监听数据的方法，而Watcher是watch的底层实现逻辑，watcher包含watch，但不止于watch。

三种Watcher

按照在Vue optionApi中的执行顺序

  • computed watcher：执行计算属性更新computed
  • user Watcher：用户注册的普通Watcher
  • render Watcher：视图更新watcher
